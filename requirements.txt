# INSTALLATION INSTRUCTIONS

#
# 1. Create virtual environment:
#    python -m venv venv
#
# 2. Activate virtual environment:
#    Windows:   venv\Scripts\activate
#    Mac/Linux: source venv/bin/activate
#
# 3. Install all requirements:
#    pip install -r requirements.txt
#
# ============================================
# PLATFORM-SPECIFIC NOTES
# ============================================
#
# WINDOWS:
# - PyAudio should install directly via pip
# - If issues, download wheel from:
#   https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio
#   Then: pip install PyAudio‑0.2.11‑cp3X‑cp3Xm‑win_amd64.whl
#
# MAC:
# - Install portaudio first: brew install portaudio
# - Then: pip install pyaudio
#
# LINUX (Ubuntu/Debian):
# - sudo apt-get update
# - sudo apt-get install python3-pyaudio portaudio19-dev
# - sudo apt-get install python3-dev
# - pip install pyaudio
#
# ============================================
# GPU ACCELERATION (OPTIONAL)
# ============================================
#
# For NVIDIA GPU support (faster processing):
#
# Instead of torch==2.1.0, install CUDA version:
# pip install torch==2.1.0+cu118 torchvision==0.16.0+cu118 --index-url https://download.pytorch.org/whl/cu118
#
# For TensorFlow GPU:
# pip install tensorflow[and-cuda]==2.15.0
#
# ============================================
# TROUBLESHOOTING
# ============================================
#
# If you get version conflicts:
# pip install --upgrade pip
# pip install -r requirements.txt --upgrade
#
# If specific package fails:
# pip install package-name --no-deps
# Then reinstall: pip install -r requirements.txt
#
# For clean reinstall:
